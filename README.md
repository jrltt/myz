# My Z

Website featuring an integrated weather API, providing users with real-time weather updates based on their location. The integration ensures an intuitive and efficient user experience when accessing a private dashboard.

This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
